# 🏀 Aplicativo da Liga de Basquete

## 📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de um aplicativo para gerenciamento e acompanhamento de uma **Liga de Basquete**, permitindo organizar equipes, jogadores, partidas e estatísticas de forma prática e eficiente.

O sistema foi projetado para oferecer uma experiência simples e intuitiva, tanto para administradores da liga quanto para usuários interessados em acompanhar os jogos e resultados.

---

## 🎯 Objetivo

O objetivo principal é criar uma aplicação funcional que permita:

* Cadastro de equipes
* Gerenciamento de jogadores
* Registro de partidas
* Exibição de resultados
* Acompanhamento de estatísticas

---

## 🎓 Contexto Acadêmico

Este projeto foi desenvolvido como **requisito obrigatório da disciplina de PDS (Projeto de Desenvolvimento de Sistemas)**, com foco na aplicação prática dos conceitos estudados ao longo do curso.

## ▶️ Como Executar o Projeto

### Backend (FastAPI)

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend (Flutter)

```bash
cd frontend
flutter pub get
flutter run
```
